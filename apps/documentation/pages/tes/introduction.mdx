# Introduction

`@elixir-cloud/tes` package provides a collection of `Web Components` for interacting with
[TES (Workflow Execution Service) API](https://github.com/ga4gh/task-execution-schemas/). It offers functionalities for listing,
creating, and deleting task runs. These components facilitate seamless integration and enhanced
user experience when working with TES.

## Goals

<Steps>

### Interact with _your_ TES

User friendly interface for TES instance.

### Extensible

Allow to swap and replace various loosely coupled features
of this component to support other/custom TES implementation.

### Styling

Allow you to style the component based on you design language and
brand identity, non programmatically! via `CSS parts` and `CSS variable`.
