# Web Components

Web Components for _your_ federated cloud infrastructures, our suite of web components are designed to be used in any web application, and are built to be
interoperable with any cloud infrastructure that implements the GA4GH standards. _Got a different implementation?_ No problem, head to our [UI solution](/ui) and
cook your own web component.
