## Overview

The `ecc-client-ga4gh-wes-runs` component displays a table of workflow runs from a GA4GH Workflow Execution Service (WES). It includes features like pagination and status filtering.

## Usage

### Basic Run List

This example shows a basic list of workflow runs.

<Tabs items={['Preview', 'React', 'HTML']}>
  <Tabs.Tab>
    <RunsPreview />
  </Tabs.Tab>

  <Tabs.Tab>
    ```tsx showLineNumbers copy filename="runs-basic.tsx"
    import React from 'react';
    import { ECCClientGa4ghWesRuns } from '@elixir-cloud/wes/react';
    import { RestWesProvider } from '@elixir-cloud/wes/providers';

    export default function RunsBasic() {
      const provider = new RestWesProvider(
        'https://weskit.anuragxd.com/ga4gh/wes/v1'
      );

      return (
        <ECCClientGa4ghWesRuns 
          provider={provider}
        />
      );
    }
    ```
  </Tabs.Tab>
  
  <Tabs.Tab>
    ```html showLineNumbers copy filename="runs-basic.html"
    <!DOCTYPE html>
    <html lang="en">
      <head>

        <ecc-client-ga4gh-wes-runs>
    ```

## Properties

| Property   | Type                      | Default | Description                                                                    |
|------------|---------------------------|---------|--------------------------------------------------------------------------------|
| `provider` | `WesProvider` | `null`  | Data provider for fetching workflow runs (see [Providers](/docs/wes/providers))  |

## Events

| Event Name             | React Event Name        | Detail Type                    | Description                                    |
|------------------------|------------------------|--------------------------------|------------------------------------------------|
| `ecc-run-selected` | `onEccRunSelected` | `{ runId: string }`        | Fired when a user clicks on a workflow run row     |
