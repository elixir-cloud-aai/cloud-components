# Providers

Data providers are responsible for fetching data from and sending data to a GA4GH Workflow Execution Service (WES) endpoint. The `@elixir-cloud/wes` package includes a `RestWesProvider` for interacting with REST-based WES APIs.

## WesProvider

`WesProvider` is an abstract base class that defines the interface for all WES data providers. To create a custom provider, you can extend this class and implement its methods.

```ts
export declare abstract class WesProvider {
    abstract getRuns(
        query: RunQuery,
        signal?: AbortSignal,
    ): Promise<TableData<Run>>;
    
    abstract getRun(
        id: string,
        signal?: AbortSignal,
    ): Promise<Run | undefined>;

    abstract postRun(
        run: Run,
        signal?: AbortSignal,
    ): Promise<Run>;
}
```

## RestWesProvider

The `RestWesProvider` is a concrete implementation of `WesProvider` that uses the `fetch` API to make requests to a RESTful WES endpoint.
