# Providers

Data providers are responsible for sending data to a GA4GH Tool Registry Service (TRS) endpoint. The `@elixir-cloud/trs-filer` package includes a `RestTrsFilerProvider` for interacting with REST-based TRS APIs.

## TrsFilerProvider

`TrsFilerProvider` is an abstract base class that defines the interface for all TRS Filer data providers. To create a custom provider, you can extend this class and implement its methods.

```ts
export declare abstract class TrsFilerProvider {
    abstract postTools(
        tool: Tool,
        signal?: AbortSignal,
    ): Promise<Tool>;
}
```

## RestTrsFilerProvider

The `RestTrsFilerProvider` is a concrete implementation of `TrsFilerProvider` that uses the `fetch` API to make requests to a RESTful TRS endpoint.
