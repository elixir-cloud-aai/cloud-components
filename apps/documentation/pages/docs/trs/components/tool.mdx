## Overview

The `ecc-client-ga4gh-trs-tool` component displays detailed information about a specific tool from a GA4GH Tool Registry Service.

## Usage

### Basic Tool Details

This example demonstrates displaying tool details for a specific tool ID.

<Tabs items={['Preview', 'React', 'HTML']}>
  <Tabs.Tab>
    <ToolPreview />
    <Callout type="info" emoji="ℹ️">
      The preview above uses a fixed `toolId`. Since the TRS instance is not production ready, this tool may not exist.
    </Callout>
  </Tabs.Tab>

  <Tabs.Tab>
    ```tsx showLineNumbers copy filename="tool-basic.tsx"
    import React from 'react';
    import { ECCClientGa4ghTrsTool } from '@elixir-cloud/trs/react';
    import { RestTrsProvider } from '@elixir-cloud/trs/providers';

    export default function ToolBasic() {
      const provider = new RestTrsProvider(
        'https://trs-filer.anuragxd.com/ga4gh/trs/v2'
      );

      return (
        <ECCClientGa4ghTrsTool 
          provider={provider}
          toolId="placeholder-tool"
        />
      );
    }
    ```
  </Tabs.Tab>
  
  <Tabs.Tab>
    ```html showLineNumbers copy filename="tool-basic.html"

    ```
  </Tabs.Tab>

## Properties

| Property    | Type                      | Default | Description                                                                    |
|-------------|---------------------------|---------|--------------------------------------------------------------------------------|
| `toolId` | `string`                  | `""`    | ID of the tool to display details for                                       |
| `provider`  | `TrsProvider` | `null`  | Data provider for fetching tool data (see [Providers](/docs/trs/providers)) |

## Events

| Event Name             | React Event Name        | Detail Type                    | Description                                    |
|------------------------|------------------------|--------------------------------|------------------------------------------------|
| `ecc-tool-version-changed` | `onEccToolVersionChanged` | `{ toolId: string, versionId: string }`        | Fired when a user selects a tool version     |
