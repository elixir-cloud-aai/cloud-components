# Providers

Data providers are responsible for fetching data from a GA4GH Tool Registry Service (TRS) endpoint. The `@elixir-cloud/trs` package includes a `RestTrsProvider` for interacting with REST-based TRS APIs.

## TrsProvider

`TrsProvider` is an abstract base class that defines the interface for all TRS data providers. To create a custom provider, you can extend this class and implement its methods.

```ts
export declare abstract class TrsProvider {
    abstract getTools(
        query: ToolQuery,
        signal?: AbortSignal,
    ): Promise<TableData<Tool>>;
    
    abstract getTool(
        id: string,
        signal?: AbortSignal,
    ): Promise<Tool | undefined>;
}
```

## RestTrsProvider

The `RestTrsProvider` is a concrete implementation of `TrsProvider` that uses the `fetch` API to make requests to a RESTful TRS endpoint.
