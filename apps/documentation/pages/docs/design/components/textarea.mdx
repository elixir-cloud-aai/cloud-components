## Overview

The `ecc-utils-design-textarea` component provides a multi-line text input field for forms, suitable for capturing longer user input.

<Callout type='info'>
  This component is a styled wrapper around the native HTML `<textarea>` element and supports disabled states, placeholder text, and resizability.
</Callout>

## Properties

| Property      | Type      | Default | Description                               |
|---------------|-----------|---------|-------------------------------------------|
| `placeholder` | `string`  | `""`    | Placeholder text for the textarea.        |
| `value`       | `string`  | `""`    | The current value of the textarea.        |
| `disabled`    | `boolean` | `false` | When `true`, disables the textarea.       |
| `rows`        | `number`  | `3`     | The number of visible text lines.         |
| `cols`        | `number`  | `undefined` | The visible width of the text control.  |

## Events

| Event Name          | React Event Name     | Detail Type             | Description                                  |
|--------------------|---------------------|-------------------------|----------------------------------------------|
| `ecc-input-changed` | `onEccInputChanged` | `{ value: string }`     | Fired when the value of the textarea changes.|

### `ecc-input-changed`
This event is fired whenever the user types or pastes content into the textarea.

<Tabs items={['React', 'JavaScript']}>
  <Tabs.Tab>
    ```tsx copy
    <EccUtilsDesignTextarea 
      onEccInputChanged={(event) => {
        console.log(event.detail.value);
      }}
    />
    ```
  </Tabs.Tab>

  <Tabs.Tab>
    ```javascript copy

    });
    ```
