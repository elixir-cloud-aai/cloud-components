## Overview

The EccUtilsDesignCode component is a customizable code editor built using the Ace editor. It provides a user-friendly interface for displaying and editing code in various languages, with syntax highlighting and theme support.

## Properties

| Property   | Required | Default      | Type      | Description                                   |
|------------|----------|--------------|-----------|-----------------------------------------------|
| `value`    | false    | `""`         | `String`  | The initial code content                      |
| `label`    | false    | `"Code block"` | `String`  | Label displayed above the code editor         |
| `language` | false    | `"json"`     | [`Language`](#language)| The programming language for syntax highlighting |
| `disabled` | false    | `false`      | `Boolean` | Whether the editor is read-only               |

### value

The `value` property sets the initial content of the code editor.

### label

The `label` property specifies the text displayed above the code editor.

### language

The `language` property determines the syntax highlighting. Supported values are:
- `"yaml"`
- `"json"`
- `"xml"`
- `"makefile"`
- `"sh"`

<Callout type='info'>
  <div className='text-sm'>
    While we maintain & provide support for the above languages, the Ace editor supports many more.
    For a full list of supported languages, refer to the [Ace editor documentation](https://ace.c9.io/).
  </div>
</Callout>

### disabled

When `true`, the `disabled` property makes the code editor read-only.
