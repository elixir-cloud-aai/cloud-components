## Overview

The `ecc-utils-design-skeleton` component is used to display a placeholder preview of content while it is loading. It provides a visual cue that a component has not yet rendered, improving the user experience by reducing perceived load times.

<Callout type='info'>
  This component is designed to be styled with standard CSS classes to match the dimensions of the content it is replacing.
</Callout>
