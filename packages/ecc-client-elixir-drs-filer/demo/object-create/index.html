DRS Object Create Demo

This demo shows the DRS Object Create component for creating new DRS objects in the DRS Filer service.

← Back to demos

Create New DRS Object

Features

  • • Create both blob and bundle DRS objects
  • • Add multiple checksums for data integrity
  • • Configure access methods (HTTP, HTTPS, S3, etc.)
  • • Support for aliases and versioning
  • • Bundle creation with nested object references

Note

This demo uses the DRS Filer API endpoint. Make sure the service is accessible from your network for object creation to work properly.
