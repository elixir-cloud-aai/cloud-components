WEB COMPONENTS - TES

1: ecc-client-ga4gh-tes-service
The ecc-client-ga4gh-tes-runs is Web Component designed to facilitate the listing of Task Execution Service service information.
2: ecc-client-ga4gh-tes-run
The ecc-client-ga4gh-tes-run component is an integral part of the ecc-client-ga4gh-tes-runs component, which is used for listing TES tasks in a paged or infinite scrolling fashion. This allows users to paginated manner. The ecc-client-ga4gh-tes-run component serves as an individual item within the task list, providing additional functionality for displaying and managing detailed information about a specific task.
The ecc-client-ga4gh-tes-runs is Web Component designed to facilitate the listing of Task Execution Service tasks in a paginated manner. It provides an intuitive interface for retrieving and displaying a collection of tasks, allowing users to efficiently navigate through a potentially large number of tasks.
3: ecc-client-ga4gh-tes-runs
4: ecc-client-ga4gh-tes-create-run
The web component ecc-client-ga4gh-tes-create-run is designed to facilitate the creation of tasks using the Task Execution Schema API provided by the Global Alliance for Genomics and Health (GA4GH). This component allows users to define various parameters and inputs required for executing tasks through TES.
