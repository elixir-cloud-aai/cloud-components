# @elixir-cloud/wes

A web component for interacting with GA4GH Workflow Execution Service (WES) API.

## Installation

```bash
npm install @elixir-cloud/wes
```

## Development

```bash
# Install dependencies
npm install

# Build the component
npm run build

# Run the demo
npm run demo

# Run tests
npm test
```
